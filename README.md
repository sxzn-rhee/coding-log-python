# coding-log-python

파이썬을 처음부터 공부하면서 **매일 기록(log)** 과 **실습 코드(snippets)** 를 쌓는 레포.

## 폴더 구조
- `logs/` : 날짜별 학습 기록 (예: `2026-01-13.md`)
- `snippets/` : 주제별 실습 코드 + 노트
  - `snippets/01_import_as_from/` : import / as / from 실습
  - `snippets/02_basics_data/` : 리스트/딕셔너리/루프/누적 등 기본 데이터 실습

## 실행 방법
레포 루트에서 아래처럼 실행:

```bash
python3 snippets/01_import_as_from/example.py
python3 snippets/02_basics_data/01_run_order.py
python3 snippets/02_basics_data/ledger_summary.py
```
