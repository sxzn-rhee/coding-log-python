# 02. 변수와 기본 자료형

## 2.1 기본 데이터 타입
정수(int), 실수(float), 문자열(str)

## 2.2 파이썬과 변수

- **변수(variable)**: 어떤 값에 대한 이름표
1. 변수 이름은 대문자와 소문자를 구분
2. 변수의 이름은 영문(한글), 숫자, 언더스코어(_)만 사용
3. 변수의 이름은 숫자로 시작 불가
- 바인딩: 변수가 자주 사용되는 어떤 값을 바인딩 -> 해당 값이 메모리에 위치하는 주소를 기억하는 것과 같음 

## 2.3 파이썬 문자열
- 문자열 인덱싱: 문자 하나하나에 인덱스라고 불리는 숫자를 맵핑

### 문자열 인덱싱 lang[숫자] 
바인딩하는 문자열 중 해당 숫자에 해당하는 글자를 가져옴

### 문자열 슬라이싱 
범위를 지정하여 문자열의 일부를 가져오는 것: 문자열[시작인덱스:끝인덱스]


## 2.4 문자열 주요 함수

### 대문자와 소문자
- upper() : 대문자로 변경하기 위한 함수 ex. lang.upper()
- lower() : 소문자로 변경하기 위한 함수

### 문자열 분리
- split() : 문자열을 구분자를 기준으로 분리할 때 사용하는 함수 
ex. 
```bash
date = "2026-01-21" 
date.split('-')
```

### 문자열 변경
- replace(바꾸기 전, 바꾸려고 하는거) 함수
ex. 
```bash
lang = "Python3"
lang2 = lang.replace('3', '4')
```

### 문자열 길이와 공백 제거 
- 문자열의 길이: len 내장 함수 
```bash
date = "2026-01-21" 
print(len(data))
```
- 공백 제거: strip()
```bash
lang = "    Python3"
lang = lang.strip()
```

## 2.5 문자열 포매팅
- %d:정수, %s: 문자열 , %f: 실수, : 형식 지정 문자열에 지정된 타입의 값이 전달되어 하나의 문자열이 됨
```bash
print("%s의 총점은 %d, 평균은 %f입니다. " % (name, score, average))
``` 
- **f-string**: ()안에 변수를 바로 사용할 수 있다. 
ex. 
```bash
name = "철수" 
score = 290
average = 290/3
print(f"{name}의 총점은 {score), 평균은 {average}입니다.")
``` 
1. 데이터가 표현될 자릿수 10칸으로 지정, 우측 정렬: {변수: >10}
2. 천 단위에서 쉼표를 표시: {변수: ,}


## 2.6 타입 변환
정수, 실수, 문자열 간의 타입 변환
ex. 
```bash
year = "2022"
year2 = int(year)

money = "1,000,000"
money2 = money.replace{',', ''}]
int(money2)
```
